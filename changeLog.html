ChangeLog

March 23, 2024

  • Fixed an issue where traits with multiple benefits could freeze the calculator(Shadeskin, Handling, etc)
  • Added missing stats for Explorer's Gold Digger ability. Note though that this means all bonuses granted by any Fountain will be considered active at all times while the ability is left toggled as on. I do not intend to add ability-bonus individual substat toggles, ability toggles themselves are already enough, use your head when abilities have more than one benefit that may not be guaranteed active at all times during ability use.

March 22, 2024

    Barring any crazy bugs you find that need immediate fixing, this is probably the last set of minor fixes to come, before I get damage calculations set up and rolling for at least melee.
  • Fixed an issue where lifesteal's display values were rounding off one digit too soon. This didn't actually change the values, only the displays(not that lifesteal is even used in any calculations front-end yet, idk why I'm even clarifying)
  • Fixed an issue where the calculator would not recognize if it was opened from another section of the site, causing it to notify you of an import when there obviously was none

March 21, 2024

  • Fixed built-in melee mods not getting cleared when the weapon was no longer selected
  • Major code overhaul: rewrote trait logic, cut out a lot of code in the process. Nothing should change on the user end, just think of it as housecleaning.

March 20, 2024

  • Added built-in gun mods. As such, imports from R2TK using these will no longer flag an exclusion notification
  • Fixed image for Dull Steel Ring
  • Finalized regex filter used for substat color tagging. I can now add to a list of exclusions specific to substat tags, for better and more efficient handling.

March 19, 2024

  • Added substat color tagging. So stuff like BULWARK or OVERLOADED, that kind of stuff now will adjust color accordingly within descriptions.
  • Improved regex filter used in the aforementioned point, to exclude compounded words or words with ":" or "-". This way fire-rate does not get modified just because it has fire in it and the hyphen wasn't recognized.
  • Added all backend statistics and custom item functions(if applicable) for the following categories. These were previously partially ignored as the skipped stats had nothing to do with tanking, EHP, or survivability in general.
    Some of these additions will reflect right now, such as Shadeskin, but a lot of these don't have displays hooked up yet.
    • Relic Fragments
    • Traits
    • Concoctions/Consumables
    • Melee Mutators
  • Added array handling for traits with multiple benefits, such as Shadeskin, or Handling. This is unique to Traits as they have levels as well as properties.
  • Added missing image for Onyx Pendulum
  • Added missing consumable descriptions, for the items that were added yesterday

March 18, 2024

  • Added custom, dynamic item functions for amulets AND rings. So like Daredevil's Charm, Gift of the Unbound, etc. Not everything will be perfect, not everything this does is even visible right now, but it's there for later.
  • Added missing consumables, even if they don't have a place yet.
  • Change settings Toggles icon to orange because people are blind and didn't know it existed. TELL ME YOU CAN'T SEE IT NOW.
  • Fixed spelling errors that did not conform to exports.

March 17, 2024

  • Fixed incorrect spelling "Hotshot" to "Hot Shot", along with its image name/path
  • Added mobile formatting to the homepage and honkai homepage
  • Added the changelog/to-do list page, which you are viewing now
  • Added changelog/to-do list page to the navbar with a clipboard icon
  • Added various backend statistics, not in use yet. These mean nothing without their respective calculations, which will come in a future update
  • Shifted repeating key CSS declarations to a new file main.css. Things like navbar/footer have been moved there for uniformity in adjustments, between all pages that use these elements.
  • Major code overhaul. Bundled functions by type, moved global vars to data.js under globalRecords
  • Added a notification that checks the query string in an import for any invalid items, and lists them so the user will know if an import error occurs that is related to spelling, or lack of an item in my data sets.

March 13, 2024

  • Reformat for the skills areas
  • Added missing ":" on mod header
  • Fixed font-size on mod/mutator selector elements
  • Fixed an issue where advanced stats relic healing was not displayed properly
  • Fixed an issue where EHP/s was not factoring in relic healing properly
  • Added basic melee statistics to data.js
  • Added display handling for the aforementioned statistics, which will now display on the rows underneath the weapon
  • Added images and descriptions for all built-in melee mods. These will now populate when the associated melee weapon is selected

March 12, 2024

  • Reformat for: Armor, Weapons, Relic, Accessory, and Prime Perk sections
  • Fixed an issue where the main body wasn't centering properly relative to the space leftover from the navbar
  • Added a tremendous amount of mobile formatting adjustments
  • Fixed an issue where mobile dropdown menus were not centering properly
  • Shifted to a fixed height on the mobile navbar, so it wouldn't constantly adjust depending on the scroll state of the page
  • Fixed an issue where the class title was too big for its container depending on the combination in use
  • Fixed "Devoured Loop" spelling, was "Devourer Loop"
  • Added handling/logic if an import was read to have higher traits spent than the 85 point limit. When such an import is received, it will read backwards up the trait list, and remove points until we reach the 85 point limit
  • Fixed an incorrect image path for Singed Ring

March 11, 2024

  • 1.0 Export Feature Collaboration with remnant2toolkit.com
  • Changed most image filepaths away from imgur links. We crutched pretty hard on imgur when the calc was just a spreadsheet, but now that we're hosting our own service, and since it was technically against imgur TOS to do what we were doing, this marks the shift towards removing all imgur links in use
  • Added the images for armor, and adjusted the filepaths away from imgur links. This should be the last of the imgur links
  • Fixed minor width issues with Firefox. Firefox is hellscape that breaks all my formatting, and there's still lots to do here in the future

March 10, 2024

  • Added the ability for external sources to import builds
  • Fixed an issue where traits were not populated correctly
  • Fixed an issue where imported build notifications were shown when not relevant.
  • Added URL item logging. Now, all selections are stored in and actively modify the URL. This means that the link can be used to "share" the build if needed.
  • Fixed an issue where URL logging did not actively modify the URL correctly
  • Fixed an issue where concoction limit was not being honored by URL logging
  • Decoded %2C back to commas within the URL logging, to reduce query string length
  • Added URL logging for toggles, and advanced stat settings. Now EVERYTHING is saved in a URL if sharing is needed in its current state, settings and all, settings no longer have to be rechecked upon pageload

March 9, 2024

  • Mobile friendly formatting v0.1. Hardly perfect, but better than nothing until I can improve it

March 8, 2024

  • Trait dropdowns now exclude what has already been selected in other boxes
  • Fixed an error that would occur if the user generated more than 20 traits. This was due to the toggles being generated based on the prior limit of 20 traits, toggles are now generated automatically according to the current amount of active traits
  • Added the trait cap functionality, matching it to the limit of 85 points like the game
  • Added smol qol feature to delete a trait if it were level 0 and the "-" adjustment was clicked. Traits at this stage will display a trash can icon instead of a "-" symbol on the button
  • Added styling and additional logic to autopopulated traits

March 7, 2024

  • Added trait autopopulation based on selected archetypes, as well as the ability to delete traits.
  • Fixed an issue when certain browser extensions modified html with injections/parse, the trait generation box would not populate correctly.

March 5, 2024

  • Updated Mutator information
  • Added corrupted weapons
  • Alphabetized selector element lists
  • Fixed an issue for when relics were selected that did not provide direct healing(eg: Lifesteal, Shields)
  • Optimized dupe/swap handling, added the same logic to quickuse, mods, and mutators
